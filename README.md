A modern, high-performance frontend landing page designed for an industrial engineering and fabrication company. This project features a premium dark-themed UI, fluid interactions, and complex scroll-triggered animations to deliver an engaging user experience.

## 🚀 Live Preview
Check out the live deployment here: **[osa-engineering.vercel.app](https://osa-engineering.vercel.app)**

## ✨ Key Features

* **Advanced Animations:** Built with GSAP (GreenSock) and ScrollTrigger for smooth, high-performance scroll reveals, staggered timeline sequences, and parallax effects.
* **Custom 3D Interactions:** Vanilla JavaScript math calculates dynamic 3D tilt effects on core pillar cards based on mouse position.
* **Pure CSS Architecture:** Fully styled using standard CSS3 without reliance on utility frameworks. Utilizes modern CSS variables, Grid, Flexbox, and fluid typography (`clamp()`) for absolute control over the design.
* **Premium Industrial UI:** A dark, high-contrast aesthetic tailored for the engineering sector, complete with custom cursors, dynamic hover states, and seamless section transitions.
* **Fully Responsive:** Flawless rendering across desktop, tablet, and mobile devices with a custom-built mobile drawer navigation.

## 🛠️ Technology Stack

* **Markup:** HTML5 (Semantic Structure)
* **Styling:** Pure CSS3 (Custom Properties, Grid, Flexbox, Media Queries)
* **Scripting:** Vanilla JavaScript (ES6+)
* **Animation Library:** GSAP (GreenSock Animation Platform) & ScrollTrigger
* **Icons & Typography:** FontAwesome 6, Google Fonts (Space Grotesk)
* **Deployment:** Vercel
